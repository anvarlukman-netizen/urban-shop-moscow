# Urban Shop Moscow — Запуск проекта

## Что нужно перед стартом
- Node.js установлен (проверь: `node -v` в терминале — должно быть v18+)
- Интернет-соединение (npm скачает пакеты)

---

## ШАГ 1 — Установить зависимости Backend

Открой терминал (PowerShell) в папке проекта и выполни по очереди:

```powershell
cd "C:\Users\Admin\Desktop\проекты ии\tg app\backend"
npm install
```

Ждёшь ~1-2 минуты пока скачаются пакеты. Должно завершиться без ошибок.

---

## ШАГ 2 — Создать таблицы в базе данных

```powershell
npm run db:push
```

Создаст файл `data/shop.db` с таблицами products, orders, favourites.

---

## ШАГ 3 — Заполнить базу тестовыми товарами

```powershell
npm run db:seed
```

Должно вывести: `Seeded 20 products`

---

## ШАГ 4 — Запустить Backend

```powershell
npm run dev
```

Должно вывести:
```
Telegram bot started
Server running on http://localhost:3001
```

**Не закрывай это окно!** Backend работает пока окно открыто.

---

## ШАГ 5 — Установить зависимости Frontend

Открой **второй** терминал (PowerShell):

```powershell
cd "C:\Users\Admin\Desktop\проекты ии\tg app\frontend"

```

---

## ШАГ 6 — Запустить Frontend

```powershell

```

Должно вывести:
```

  Local:   http://localhost:5173/
```

## ШАГ 7 — Открыть в браузере

Перейди по адресу: **http://localhost:5173**

Ты увидишь магазин. В браузере работает в режиме разработки (без Telegram — данные пользователя будут тестовыми).

## Если что-то пошло не так

### Ошибка при `npm install` (backend)
Попробуй:
```powershell
npm install --legacy-peer-deps
```

### Ошибка `Cannot find module`
```powershell
npm install
```

### Порт 3001 занят
В файле `backend/.env` измени `PORT=3001` на `PORT=3002`, и в `frontend/src/api/client.ts` измени URL с 3001 на 3002.

### База данных пустая / ошибка при seed
```powershell
npm run db:push
npm run db:seed
```

## Деплой в интернет (когда готов к запуску)

### Frontend → Vercel (бесплатно)

1. Зайди на [vercel.com](https://vercel.com), зарегистрируйся
2. Нажми "Add New Project"
3. Перетащи папку `frontend` или подключи GitHub
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Нажми Deploy

Получишь адрес вида `https://urban-shop-xxxx.vercel.app`

### Backend → Railway

1. Зайди на [railway.app](https://railway.app), зарегистрируйся
2. New Project → Deploy from GitHub / Upload
3. Загрузи папку `backend`
4. В разделе Variables добавь все переменные из `.env`
5. Railway автоматически запустит `npm start`

Получишь адрес вида `https://urban-shop-backend.up.railway.app`

### После деплоя

В `frontend/src/api/client.ts` замени:
```
const BASE_URL = 'http://localhost:3001';
```
на:
```
const BASE_URL = 'https://твой-backend.up.railway.app';
```

### Регистрация Mini App в Telegram

1. Напиши @BotFather в Telegram
2. Команда `/newapp`
3. Выбери бота `@UrbanShopMoscowBot` (или как назвал)
4. Вставь URL фронтенда: `https://urban-shop-xxxx.vercel.app`
5. Готово — ссылка на Mini App появится в профиле бота

## Команды бота (для менеджера)

Напиши боту в Telegram:
- `/orders` — последние 10 заказов
- `/orders_new` — только новые заказы
- `/stats` — статистика (сегодня / всего)

При новом заказе бот сам пришлёт уведомление с кнопками "Подтвердить" / "Отменить".
